The difference between Linear Search and Binary Search

Linear Search
1. Starts at the first element and traverses the list sequentially
2. Requires both sorted and usorted data
3. Small , datasets or unsorted collections

Binary Search
1. Compare the target to the middle element ,narrowing the search rangr
2. Requires only sorted data
3. large, sorted data
